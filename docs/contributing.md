# Contributing

We welcome contributions! Please follow these guidelines:

## How to Contribute
- Fork the repo and create your branch from `development`
- Write clear, descriptive commit messages
- Add/maintain tests for your changes
- Run linting and formatting before pushing
- Open a PR and fill out the template

## Code of Conduct
- Be respectful and inclusive
- No harassment, discrimination, or toxic behavior
- See [Contributor Covenant](https://www.contributor-covenant.org/)

## Issue & PR Templates
- Use the provided templates for bug reports and feature requests
- Include steps to reproduce, expected/actual behavior, and screenshots if relevant
